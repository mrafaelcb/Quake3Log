# Quake3Log
Criar Virtualenv
```
  python -m venv myvenv
```
Ativar Virtualenv
```
  myvenv\Scripts\activate
```
Atualizar pip
```
  python -m pip install --upgrade pip
```
Instalar requisitos
```
  pip install -r requirements.txt
```
rodar servidor
```
  python manager runserver
```
Servidor
http://127.0.0.1:8000
